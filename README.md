# Task Manager API

REST API built with FastAPI and SQLAlchemy using SQLite.

## Features

- CRUD operations
- SQLAlchemy ORM
- SQLite persistence
- Dependency injection
- Automatic Swagger documentation

- - JWT authentication
- OAuth2 login flow
- Password hashing with bcrypt
- Protected user-specific task access
- Owner-based authorization

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- JWT (python-jose)
- Passlib (bcrypt)

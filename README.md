# Task Manager API

REST API built with FastAPI and SQLAlchemy using SQLite.

## Features

- CRUD operations
- SQLAlchemy ORM
- SQLite persistence
- Dependency injection
- Automatic Swagger documentation

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Pydantic

1. Create virtual environment
2. Install dependencies
